# Блосинданглиф (Blosindanglif)

Добро пожаловать в официальный репозиторий проекта **Блосинданглиф**! 🚀

Здесь хранится база нашего уникального языка иероглифов, его алфавит, правила и первые попытки записи слов. 

## Что это за язык?
Блосинданглиф — это система иероглифов, созданная для шифровки и общения. Это не просто алфавит, это наш собственный способ кодирования мыслей.

## Структура репозитория
* `/assets` — графические файлы с алфавитом и ключами расшифровки.
* `index.html` — официальная веб-страница проекта.

## Как пользоваться?
Используй таблицы в папке `/assets` для обучения. Если хочешь помочь с развитием языка или сайта, делай форк и предлагай свои изменения!
