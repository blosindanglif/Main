# Ассеты Блосинданглифа

В этой папке хранятся все визуальные материалы проекта **Блосинданглиф**.

## Что внутри:
* **Алфавит**: Графическая таблица соответствия иероглифов буквам.
* **Примеры**: Изображения с зашифрованными словами.

Эти файлы являются ключом к пониманию нашего языка. Используйте их для расшифровки и обучения.
